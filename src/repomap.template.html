How to create a token

Fine-grained token (recommended):

  1. Go to Fine-grained tokens
  2. Select repositories: All repositories
  3. Under Repository permissions, grant Metadata — Read-only
  4. Click Generate token and paste it above

Classic token:

  1. Go to New classic token
  2. Select scope: read:user (add repo for private repos)
  3. Click Generate token and paste it above
